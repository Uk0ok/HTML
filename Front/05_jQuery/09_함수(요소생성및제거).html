삽입 관련 메소드 part1

새로운 요소 추가하는 메소드

$(A).append(B)

A 요소 내부의 뒷부분에 B 추가 (자손)

$(A).prepend(B)

A 요소 내부의 앞부분에 B 추가 (자손)

$(A).after(B)

A 요소 뒷부분에 B 추가 (형제) ex) div 밖에서 다음

$(A).before(B)

A 요소 앞부분에 B 추가 (형제) ex) div 밖에서 이전

위의 예제와 A, B 자리가 바뀜

$(B).appendTo(A)

$(B).prependTo(A)

A 요소 내부의 앞부분에 B 추가 (자손)

$(B).insertAfter(A)

A 요소 뒷부분에 B 추가 (형제)

$(B).insertBefore(A)

A 요소 앞부분에 B 추가 (형제)

요소객체 복제 메소드

변수 = $("선택자").clone(true|false) true|false : 복제할 요소의이벤트까찌 복사할건지? (default:false)

요소 객체 제거 및 잘라내기 함수

$("선택자").empty() : 하위 요소들 제거

$("선택자").remove|detach() : 제거 후 해당 요소 반환 (잘라내기)
remove vs detach : 이벤트 가져오는지 여부 (이벤트 가져오려면 detach)
