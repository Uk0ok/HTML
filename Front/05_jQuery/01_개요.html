jQeury 개요

jQeury 란?

좀 더 편하게 작업하려고 jQeury를 배운다.
js를 이용해서 라이브러리를 만들어 놓은 것

복잡한 자바스크립트 구문을 짧고 편하게 작성 가능

jQeury 장점

1) DOM 조작 쉬움
2) AJAX 지원
3) 플러그인, api 지원 해주는 거 많음(차트, 슬라이드 등등...)

jQeury 라이브러리 연결 방법

1) 파일
2) 링크 (CDN 방식)
